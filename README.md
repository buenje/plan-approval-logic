# Lernpfad Solidity – 14 Tage (bunje_blockchain)

**Ziel:** sichere Basis von Mitte Level 1 → Anfang Level 2 in 14 Tagen, mit minimalem Code und maximaler Klarheit.  
**Rhythmus:** 12–16 Stunden/Woche (täglich 60–120 Min), Fokus: *sehen → klicken → Mini‑Edit*.

**Inhalte:** Siehe [`learnpath/LEARNPFAD_14_TAGE.md`](learnpath/LEARNPFAD_14_TAGE.md) und die Checkliste [`learnpath/CHECKLISTE.md`](learnpath/CHECKLISTE.md).

---

## Schnellstart (GitHub)
1. Dieses Repo lokal entpacken/ablegen.
2. Terminal:
   ```bash
   git init
   git add .
   git commit -m "init: 14-Tage-Lernpfad Solidity"
   git branch -M main
   git remote add origin https://github.com/<dein-user>/<dein-repo>.git
   git push -u origin main
   ```

## 5-Min-Demo (Remix)
- Öffne deinen Übungsvertrag (z. B. FreigabeNotizbuch).
- Klicke `pause()/unpause()`, lies `paused()` (blauer Button = kein Chain‑Eintrag).
- Füge 1 Mini‑Edit ein (z. B. `isPaused()` als `view`) → kompilieren → 1 Screenshot.
- Schreibe **einen Satz**: *„Was habe ich gesehen/warum?“*
